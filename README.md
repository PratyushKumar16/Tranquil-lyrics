# Tranquil Lyrics


# How to install Tranquil Lyrics

## 1. Using the Spicetify Marketplace (recommended)
1. Search `Tranquil Lyrics` under the "Extensions" tab
2. Click the Install button on the Tranquil Lyrics extension
3. All done!

## 2. Externally (not recommended)
1. Make sure you have [Spicetify](https://spicetify.app) installed
2. Download the [tranquil-lyrics.mjs](./builds/tranquil-lyrics.mjs) file
3. Put the file inside the Spicetify Extensions directory. Find the correct directory here: [https://spicetify.app/docs/customization/extensions#manual-installation](https://spicetify.app/docs/customization/extensions#manual-installation)
4. Then, run ```spicetify config extensions tranquil-lyrics.mjs```
5. Then apply Spicetify by running ```spicetify apply```
6. All done!

I noticed a major problem with other real-time Spotify lyrics extensions like Beautiful Lyrics and Spicy Lyrics: they are highly unoptimized and drain excessive CPU power and battery.

I built **Tranquil Lyrics** to address this unoptimization issue, providing a beautiful and dynamic real-time lyrics experience while being extremely lightweight and resource-friendly on your machine. Hope you like it!
